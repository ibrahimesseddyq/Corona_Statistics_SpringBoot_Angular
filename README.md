# API's urls:

## Get users:
[GET] /api/users/
## Get specific user by id:
[GET] /api/users/{id}
## Get specific user by username:
[GET] /api/users/username/{username}
## Save user:
[POST] /api/users/
## Get roles:
[GET] /api/users/roles
## Save role:
[POST] /api/users/role/save
## Add role to user :
[POST] /api/users/role/addtouser
## Login :
[POST] /login
